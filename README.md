# Simple Spring MVC App
Foundation for other Spring MVC projects, features:
- Spring MVC 4.x with code config
- WebApplicationInitializer, WebMvcConfigurerAdapter
- no WEB.xml
- async request processing enabled
- tested on Jetty 9
